# 📚 内容参考与迭代记录

这个目录用来沉淀内容优化过程中的：
- **优秀案例**：来自小红书、公众号等平台的高质量内容参考
- **用户反馈**：你对内容的批评、建议、方向调整
- **迭代记录**：每次改了什么、为什么改、效果如何

## 目录结构

```
references/
├── README.md              ← 本文
├── cases/                 ← 优秀案例（截图/文案/链接）
│   └── (抓取自优秀博主的内容片段)
├── feedback/              ← 你的反馈意见
│   └── 2026-06-15-首篇反馈.md
└── changelog.md           ← 迭代变更记录
```
